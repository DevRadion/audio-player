# audio-player
C++ Audio Player with GUI written on .NET Framework

## Used libs
- [libsamplerate](https://github.com/libsndfile/libsamplerate)
- [libsndfile](https://github.com/libsndfile/libsndfile)
- [PortAudio](https://github.com/PortAudio/portaudio)
- WindowsAPI
